# RoboSim (Robot Simulator)

Java based portable simulator to visualize and understand the *Robot Localization*, *Path planning*, *Path Smoothing* and *PID controller* concepts.

### Contents
* Filters
  * Histogram Filters
  * Kalman Filters
  * Particles Filters
* Path Planning
* Path Smoothing
* PID Controller

### Path Planning Algorithms

* BFS
* DFS
* A Star
* Dynamic Programming

### Heuristics

* Euclidean Distance
* Euclidean Distance(+)
* Euclidean Distance(*)
* Euclidean Distance Squared
* Manhattan Distance
* Chebyshev Distance
